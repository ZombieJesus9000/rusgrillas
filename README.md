## rusgrillas

A Bevy 0.18 port of the classic QBASIC *Gorillas* game, using:

**rusgrillas** is a small Rust homage to IBM's classic QBASIC game **GORILLA.BAS** \([original source](https://gist.githubusercontent.com/caffo/1326838/raw/65460506dd08eed5605eb84405a0d0e71aac9150/GORILLA.BAS)\).

Two pixel gorillas stand atop a procedurally generated city skyline and take turns lobbing explosive bananas at each other. Gravity and wind affect each shot, and the skyline updates every run. A simple day/night cycle slowly shifts the sky color while you play.


This is a work in progess... You have been warned.
